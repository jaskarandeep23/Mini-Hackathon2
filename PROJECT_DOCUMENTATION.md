# Project Documentation

## Project Overview

This project is a **responsive portfolio-style webpage** created for the **Module 02 Mini Hackathon â€“ Responsive CSS Webpage**.  
It is built using **only HTML and CSS** (no JavaScript) and demonstrates:

- CSS selectors and properties  
- The box model, display, and positioning

- Responsive design using media queries
