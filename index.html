About Our Project

This responsive portfolio webpage was created as part of the Module 02 Mini Hackathon. Our goal was to build a clean, modern layout using only HTML and CSS, while applying concepts such as the box model, flexbox, grid, and responsive design.

The page highlights our teamwork, individual strengths, and the CSS concepts we practiced. Each group member was responsible for specific sections and styling areas, which are clearly identified below in the Team Contributions section.
